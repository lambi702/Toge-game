Avant dernière étape avant la fin
## Comment jouer?
Pour cette étape, il va falloir lancer un mini jeu java. Pour ça c'est assez simple, 
1. cloner le repos ou télécharger le zip du repo et le décompresser.
2. ouvrir un terminal dans le dossier ou se trouve TogeGame.class
3. entrer la ligne de commande : java TogeGame
